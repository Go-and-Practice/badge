##Badge.Kloud51.com

Badge/Pins/Shields/Medal or what so ever for projects.

Services Supported:

* Python [PyPi](https://pypi.python.org/)
* ArchLinux [AUR](https://aur.archlinux.org/)


### Badges

PyPi badges are the first of course, this project has been written in Python.
